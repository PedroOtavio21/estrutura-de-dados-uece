# Entregável da disciplina (Estruturas de Dados)

Este repositório contém as 20 resoluções do primeiro entregável de Estrutura de Dados, organizadas de forma clara para facilitar a correção e navegação do professor.

## Estrutura das Pastas

As resoluções são divididas da seguinte forma:

```bash
questao-<número-da-questão>/
```

Dentro de cada pasta:

Se a questão exigir a implementação de uma estrutura de dados, o arquivo estará nomeado conforme o nome da estrutura, por exemplo:

```bash
ListaEncadeada.js
ListaDuplamenteEncadeada.js
ArvoreBinaria.js
```

Se a questão pedir apenas uma implementação direta (sem necessidade de uma estrutura nomeada), o arquivo será sempre:

```bash
app.js
```

## Observações Gerais

- Todo o código está em JavaScript, utilizando classes para representar estruturas de dados.

- As implementações seguem o enunciado fornecido.

- Nenhuma biblioteca externa é utilizada.
