# Entregáveis da Disciplina — Estrutura de Dados

Este repositório contém as resoluções dos **entregáveis da disciplina de Estrutura de Dados**, organizadas de forma padronizada para facilitar a navegação, compreensão e correção.

Ao todo, o repositório é composto por **4 entregáveis**, cada um contendo implementações de estruturas de dados ou soluções de problemas específicos, conforme proposto nos enunciados da disciplina.

---

## Estrutura do Repositório

Cada entregável possui sua própria pasta, identificada pelo número correspondente:

```bash
entregavel-01/
entregavel-02/
entregavel-03/
entregavel-04/
```

Dentro de cada entregável, as soluções estão organizadas por questão.

### Estrutura interna de um entregável

```bash
entregavel-XX/
├── questao-01/
├── questao-02/
├── ...
└── questao-20/
```

---

## Estrutura das Questões

Cada pasta de questão contém **apenas os arquivos necessários para sua resolução**, seguindo o padrão abaixo:

### Questões com implementação de estruturas de dados

Quando a questão exige a implementação de uma estrutura de dados específica, o arquivo é nomeado de acordo com a estrutura implementada, por exemplo:

```bash
ListaEncadeada.js
ListaDuplamenteEncadeada.js
ArvoreBinaria.js
```

### Questões com implementação direta

Quando a questão não exige uma estrutura de dados nomeada, a solução está contida no arquivo:

```bash
app.js
```

---

## Observações Gerais

* Todo o código foi desenvolvido em **JavaScript**.
* As estruturas de dados são implementadas utilizando **classes**.
* As soluções seguem fielmente os enunciados fornecidos na disciplina.
* **Nenhuma biblioteca externa** foi utilizada.
* O padrão de organização é mantido em todos os entregáveis.

---

Este formato visa garantir clareza, organização e padronização, facilitando tanto o desenvolvimento quanto a correção dos entregáveis.
